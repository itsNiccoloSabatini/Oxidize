## ğŸš€ Getting Started

### Prerequisites

- Rust (latest stable version)
- Cargo (comes with Rust)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/itsNiccoloSabatini/Oxidize.git
cd Oxidize
```

2. Build and run:

```bash
cargo run
```

### Web Version

To run in a web browser:

```bash
trunk serve
```
